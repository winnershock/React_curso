REPOSITORIO GITHUB

Iniciar repositorio local:
"git init"

cambiar el nombre de master a main:
"git Branch -M main"

Eliminar credenciales de otros usuarios:
win + s 
Escribir: Administrador de credenciales
Seleccionar credenciales de windows y luego buscar las cuentas Github asociadas y elimina las credenciales.
Luego de eliminar las credenciales

Configurar usuario local de Git:
"git config user.name" "suNombre"
"got config user.email" "suEmail"

Ahora agregar todos los archivos del proyecto:
"git add ."

Realizar un commit descriptivo de la version a subir:
"git commit -m "Primera Subida"

Enlazar el repositorio de la nube con el local
git remote add origin https://
git push -u origin main

Luego saldra que debe iniciar sesion con el navegador:
inicie sesion con sus credenciales del Github.
listo actualiza el repositorio de la nube.


Comprobar el estado del proyecto:
git status


(si aparece algo en Rojo significa que el archivo es nuevo o se ha modificad
o).

Comandos para actualizar el repositorio con la version local:
git status
git add .
git commit -m "descripcion del commit"
git push

Crear Ramas:
git checkout -b nombre_de_la_rama
git switch -c nombre_de_la_rama
git push origin nombre_de_la_rama


cambiar a la rama main
git checkout main
